# User Instrument Screen

This screen allows the user to list and search the instruments currently known in the system.  There should be an alphabetically sorted paginated list which defaults to the first page of all of the instruments in the system.

A text field allows the user to type a search term which is matched against the display name of the instrument (ticker, name, or broker description).  The search box should function as a 'search as you type' box.

Each result should show primarily either the ticker, if available, or the broker description for the instrument.  It should then show details of other identifiers available for the same instrument as well as the instrument metadata.
